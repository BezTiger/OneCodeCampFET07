** to check git version
git --version

Configure Git, return nothing means success
Command: git config --global user.name "username"
Command: git config --global user.email "email"

Initialize an empty repository
git init

Check for the status of files
Command: git status

Add files to the staging area
Command: git add .

Create a commit
Command: git commit -m "This is the description."

Branching

List out all of the branches
Command: git branch -a

Create a branch
Command: git branch [branch-name]

Switch to a different branch
Command: git switch [branch-name]
git merge [branch-name]s
